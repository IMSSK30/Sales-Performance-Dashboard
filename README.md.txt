## Project Overview

This is my first end-to-end Data Analytics project.  
In this project, I worked on a retail sales dataset and created an interactive Sales Performance Dashboard using Excel, MySQL, and Power BI.

The main purpose of this project was to understand how raw data can be cleaned, analyzed, and converted into useful business insights.

---

## Tools Used

- Excel
- MySQL
- Power BI

---

## Steps I Followed

### Data Cleaning (Excel)

- Removed duplicate records  
- Checked missing values  
- Fixed date formats  

### Data Analysis (MySQL)

- Total Sales  
- Total Profit  
- Total Orders  
- Top Customers  
- Sales by Category  
- Sales by Region  
- Monthly Sales Trend  
- Delivery Time Analysis  

### Dashboard (Power BI)

- KPI cards  
- Sales trend charts  
- Category and region analysis  
- Customer insights  
- Slicers for filtering data  

---

## Key Insights

- Technology category performed the best in sales  
- West region had the highest sales  
- Consumer segment contributed the most revenue  
- Sales change over months showing seasonal pattern  
- Few customers contribute a large share of sales  

---

## Conclusion

This project helped me understand the complete workflow of data analysis using Excel, SQL, and Power BI.  
It improved my skills in data cleaning, analysis, and dashboard building.
