# Sales Performance Dashboard

## Project Overview

This is my first end-to-end Data Analytics project. 
In this project, I worked on a retail sales dataset and created an interactive Sales Performance Dashboard using Excel, MySQL, and Power BI.

The main purpose of this project was to understand how raw data can be cleaned, analyzed, and converted into useful business insights.

---

## Tools Used

- Microsoft Excel
- MySQL
- Power BI

---

## Steps I Followed

### 1. Data Cleaning (Excel)

- Opened the dataset in Excel.
- Removed duplicate records.
- Checked for missing values.
- Verified the date format.
- Saved the cleaned dataset.

### 2. Data Analysis (MySQL)

After importing the dataset into MySQL, I converted the date columns into the correct format using SQL.

Then I wrote SQL queries to find:

- Total Sales
- Total Profit
- Total Orders
- Average Sales
- Total Customers
- Profit Margin
- Sales by Category
- Sales by Region
- Sales by Segment
- Profit by State
- Top 10 Customers
- Top 10 Products
- Monthly Sales Trend
- Delivery Time

### 3. Dashboard Creation (Power BI)

I imported the cleaned data into Power BI and created an interactive dashboard.

The dashboard includes:

- KPI Cards
- Sales Trend
- Sales by Category
- Sales by Region
- Sales by Segment
- Top Customers
- Top Products
- Profit by State
- Slicers for Region, Category, Segment and Year

---

## What I Learned

This project helped me understand the complete workflow of a Data Analyst.

I learned how to clean data, write SQL queries, create KPIs, use DAX measures, and build an interactive dashboard in Power BI.

I also learned that data analysis is not only about creating charts. 
It is about understanding the data and finding useful information that can help in making business decisions.

---

## Key Insights

- Technology category generated the highest sales.
- The West region had the highest sales compared to other regions.
- The Consumer segment contributed the largest share of sales.
- Some customers contributed much more sales than others.
- Sales changed from month to month, showing different business trends during the year.
- The profit margin is lower than the sales growth, which shows that profit should also be monitored along with sales.
